# Security Rules

- Never include API keys, tokens, passwords, or secrets in documents or output
- Never log or display sensitive partner contact information (personal emails, phone numbers) unless explicitly provided by the user
- Sanitize any PII before including in research outputs
- When accessing web sources, do not submit forms or create accounts
- Do not share proprietary Further information in web searches or prompts to external services
- Treat all partnership discussions as confidential until told otherwise
